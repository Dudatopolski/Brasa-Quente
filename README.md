## ✨ Visão Geral

O **Brasa Quente** é um protótipo de site institucional e funcional que simula a experiência completa de um sistema real utilizado por clientes de uma churrascaria moderna.

O projeto foi pensado para demonstrar **organização**, **experiência do usuário** e **fluxos reais de navegação**, indo além de um layout estático.

---

## 🎯 Objetivo do Projeto

- Simular um site completo de uma churrascaria moderna  
- Desenvolver fluxos reais de usuário (login, cadastro e reservas)  
- Praticar organização de telas e arquivos  
- Demonstrar pensamento de produto  
- Criar um projeto forte para portfólio profissional  

---

## 🧩 Funcionalidades Simuladas

⚠️ *Este projeto é um protótipo front-end. As funcionalidades não possuem persistência real de dados.*

### 🔐 Autenticação
- Tela de login  
- Tela de cadastro  
- Recuperação de senha (simulada)  
- Validação visual de campos  

### 📅 Reservas
- Formulário de reserva de mesa  
- Seleção de data, horário e quantidade de pessoas  
- Feedback visual de confirmação  

### 📖 Cardápio
- Organização por categorias  
- Cards contendo:
  - Imagem
  - Descrição
  - Preço  
- Layout responsivo  

### 🏠 Sobre a Casa
- Cards institucionais  
- Diferenciais da churrascaria  
- Conteúdo voltado à identidade da marca
